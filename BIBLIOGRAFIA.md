## Podsumowanie

Projekt **Opiekus** wykorzystuje nowoczesny stack technologiczny oparty na:
- **Frontend**: Next.js 16, React 19, TypeScript 5
- **Stylowanie**: Tailwind CSS 4, Material-UI 7, Radix UI
- **Baza danych**: PostgreSQL z Prisma ORM 6
- **Walidacja**: Zod 4
- **Zarządzanie stanem**: TanStack Query 5
- **Uwierzytelnianie**: JWT (jose), bcryptjs
- **Upload plików**: UploadThing 7
- **Komunikacja**: Nodemailer, Web Push
- **Animacje**: Framer Motion 12
- **Wykresy**: Recharts 2

Wszystkie biblioteki są aktualne i wykorzystują najnowsze wersje dostępne w momencie tworzenia projektu.
